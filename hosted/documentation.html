/getBooks Endpoint

Description: Returns an array of json book objects, containing all of the books within the dataset.

Accepted Request Type(s): GET or HEAD

Format:

  • Query Parameters
    • None
  • Request - /getBooks
  • Result - [{"author": "Author", "country": "Country", "language": "Language", "link": "Link", "pages": number, "title": "Title", "year": number, "genres": ["genre"], "review"?: number}]

/getTitle Endpoint

Description: Returns an array of json book objects containing all of the books with a given title. Each book in the dataset must have a unique title, so the array will only ever contain 0 or 1 book.

Accepted Request Type(s): GET or HEAD

Format:

  • Query Parameters
    • title (string): The name of the book
  • Request - /getTitle?title=title1
  • Result - [{"author": "Author1", "country": "Country1", "language": "Language1", "link": "Link1", "pages": 734, "title": "Title1", "year": 1876, "genres;: ["Genre1","Genre2"]}]

/getAuthor Endpoint

Description: Returns an array of json book objects containing all of the books by a given author.

Accepted Request Type(s): GET or HEAD

Format:

  • Query Parameters
    • author (string): The author of the book
  • Request - /getAuthor?author=author1
  • Result - [{"author": "Author1", "country": "Country1", "language": "Language1", "link": "Link1", "pages": 865, "title": "Title1", "year": 1876, "genres": ["Genre1","Genre2"]}]

/getYear Endpoint

Description: Returns an array of json book objects containing all the books published in a given year or year range. The year should be positive if it is AD or negative if it is BC.

Accepted Request Type(s): GET or HEAD

Format:

  • Query Parameters
    • year (number): The year a book was published or the starting year of a range in which a book was published
    • end (number)(optional): The ending year of a range in which a book was published. Must be after the start year
  • Request - /getYear?year=1845 or /getYear?year=1800&end=1850
  • Result - [{"author": "Author1", "country": "Country1", "language": "Language1", "link": "Link1", "pages": 236, "title": "Title1", "year": 1845, "genres": ["Genre1"], "review": 6.5}]

/getGenre Endpoint

Description: Returns an array of json book objects containing all the books of a specific genre or combination of genres.

Accepted Request Type(s): GET or HEAD

Format:

  • Query Parameters
    • genre (string): The genre or genres of a book in a comma-separated list.
  • Request - /getGenre?genre=Genre1 or /getGenre?genre=Genre1,Genre2
  • Result - [{"author": "Author1", "country": "Country1", "language": "Language1", "link": "Link1", "pages": 236, "title": "Title1", "year": 1845, "genres": ["Genre1","Genre2"]}]

/getLanguage Endpoint

Accepted Request Type(s): GET or HEAD

Format:

  • Query Parameters

  • Request - /getLanguage?language=language1
  • Result - [{"author": "Author1", "country": "Country1", "language": "Language1", "link": "Link1", "pages": 865, "title": "Title1", "year": 1876, "genres": ["Genre1"], "review": 2}]

/addBook Endpoint

Description: Adds a book with a provided title, author, publication date, language, and given genres to the dataset or modifies the data of the book with the given title, if one already exists. This data is not persisted and will not be saved after the session ends.

Accepted Request Type(s): POST

Supported Body Formats: application/json and application/x-www-form-urlencoded

Format:

  • Body Parameters
    • title (string): The name of the book
    • author (string): The author of the book
    • language (string): The language the book was originally written in
    • year (number): The year the book was published
    • genres (string[]): The genre or genres of a book in a comma-separated array
  • Request - /addBook
  • Request Body - {"author": "Author1", "language": "Language1", "title": "Title1", "year": 1845, "genres": ["Genre1","Genre2"]}
  • Result - {"author": "Author1", "language": "Language1", "title": "Title1", "year": 1845, "genres": ["Genre1","Genre2"]}

/addReview Endpoint

Description: Adds a numerical rating to a book that already exists within the dataset. This data is not persisted and will not be saved after the session ends.

Accepted Request Type(s): POST

Supported Body Formats: application/json and application/x-www-form-urlencoded

Format:

  • Body Parameters
    • title (string): The year a book was published or the starting year of a range in which a book was published
    • review (number): A numerical rating of the book from 0-10 with increments of 0.5
  • Request - /addReview
  • Request Body - {"title": "Title1", "review": 5.5}
  • Result - {"author": "Author1", "country": "Country1", "language": "Language1", "link": "Link1", "pages": 865, "title": "Title1", "year": 1876, genres: ["Genre1"], "review": 5.5}
