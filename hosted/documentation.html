Book API Documentation

Click here to test the endpoints

/getBooks Endpoint

Description: Returns a json object containing all of the books within the dataset.

Accepted Request Type(s): GET or HEAD

Format: C

/getTitle Endpoint

Description: Returns a json object containing all of the books with a given title. Each book in the dataset must have a unique title, so only 1 or 0 books will ever be returned.

Accepted Request Type(s): GET or HEAD

Format: C

/getAuthor Endpoint

Description: Returns a json object containing all of the books with a given author.

Accepted Request Type(s): GET or HEAD

Format: C

/getYear Endpoint

Description: Returns a json object containing all the books published in a given year or year range.

Accepted Request Type(s): GET or HEAD

Format: A year should be positive if it is AD or negative if it is BC.

/getGenre Endpoint

Description: Returns a json object containing all the books of a specific genre or combination of genres.

Accepted Request Type(s): GET or HEAD

Format: C

/getLanguage Endpoint

Description: Returns a json object containing all books written in a given language.

Accepted Request Type(s): GET or HEAD

Format: C

/addBook Endpoint

Description: Adds a book with a provided title, author, publication date, language, and given genres to the dataset. This data is not persisted and will not be saved after the session ends.

Accepted Request Type(s): POST

Format: C

/addReview Endpoint

Description: Adds a numerical rating to a book that already exists within the dataset. This data is not persisted and will not be saved after the session ends.

Accepted Request Type(s): POST

Format: C
